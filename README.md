# Calculadora em Java

Calculadora desenvolvida com o objetivo de praticar lógica de programação e conceitos fundamentais da linguagem Java.

## Objetivo
Projeto desenvolvido em Java com foco na aplicação prática de lógica de programação, voltado à criação de uma calculadora para apoio na precificação de projetos no meu trabalho.

## Funcionalidades
- Soma
- Subtração
- Multiplicação
- Divisão

## Tecnologias utilizadas
- Java

## Aprendizados
- Estrutura básica em Java
- Lógica de programação
- Manipulação de entrada e saída
- Operações matemáticas

---

Projeto com fins educacionais.
